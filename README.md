# SheCanCode-3 Tailwind Assignment

## Description

A simple, responsive Tailwind CSS preview page built for the SheCanCode assignment. It demonstrates utility classes, offline independence, and mobile-first design while following the provided template structure.

## Features

- **Responsive Layout**: Hero section maintains 90% width requirement with mobile-first breakpoints
- **Offline Independence**: No external CDN calls — uses local `styles.css` for all styling
- **Clean Navigation**: Top bar with logo and "Get Started" / "How it Work" buttons matching the example
- **Local Images**: Images stored in `images/` folder and loaded without external dependencies
- **Mobile-First Design**: Layout adapts to small screens without horizontal scrolling
- **Code Cleanliness**: Organized CSS classes and readable HTML structure

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Niyigenaclaire/SheCanCode-3-tailwind.git
   ```

2. Navigate to the project folder:
   ```bash
   cd SheCanCode-3-tailwind
   ```

3. Open `index.html` in your web browser.

No server or build process required — it's a static HTML page.
